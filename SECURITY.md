# Security Policy

## Supported Version

Version 0.1.7 is the currently supported public release. Security fixes may also
be developed on the active development branch, but unreleased work should not be
treated as a supported public release.

| Version | Supported |
| --- | --- |
| 0.1.7 | Yes |
| Earlier versions | No |

## Report a Vulnerability

Please report suspected vulnerabilities privately. If GitHub Private
Vulnerability Reporting is enabled for this repository, use the repository's
**Security** tab and select **Report a vulnerability**. When GitHub offers that
private reporting option, include:

- The affected version and component.
- Steps to reproduce the issue.
- The security impact you believe is possible.
- A minimal proof of concept, logs, or screenshots with sensitive data removed
  where practical.
- Any suggested mitigation, if known.

If the repository does not offer a private reporting option, do not post
exploitation details, secrets, personal data, or other sensitive information in
a public issue. Use an available private GitHub reporting or contact option for
the repository owner and share only enough publicly to request a private
channel.

Do not include real dealership customer data, authentication material, or
unredacted captured report data unless it is necessary to demonstrate the issue
and can be transmitted privately.

## Scope

Security reports should concern vulnerabilities in the extension or its
repository, such as unsafe handling of captured website-controlled content,
unauthorized exposure or modification of locally stored scan data, or misuse of
extension permissions.

Privacy, cookie, consent, advertising, analytics, or tracking observations made
by the extension on a dealership website are not automatically security
vulnerabilities in the extension itself. The project reports observable browser
evidence; it does not provide legal advice, certify compliance, prove script
execution or blocking, or establish that a third-party website is vulnerable.

Reports will be evaluated based on the available evidence and project scope. No
guaranteed response or resolution time is promised.
