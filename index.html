Firing order

See the exact sequence that scripts load on your website. Your consent manager must load first — before Google, Facebook, or any tracking tool.

🍪

Cookie classification

Every cookie categorized as Essential, Analytics, or Targeting. Targeting cookies must not be set before visitor consent — see exactly what's happening.

📦

GTM container detection

Identifies every Google Tag Manager container on your site. Multiple containers are common on dealership sites and each one is a potential compliance gap.

📊

Google Consent Mode signals

Checks for gcs, gcd, npa=1, pscdl=denied, and ads_data_redaction signals — then separates strong proof from indicators that still need confirmation.

👤

Meta / Facebook pixel

Counts exactly how many Facebook requests fired in your session. The Meta pixel has no restricted mode — it either fires or it doesn't.

🌐

Third-party domains

Complete list of every external domain your website contacted during your visit — Google, Meta, CRM platforms, chat tools, and more.

Why this matters

The banner isn't enough.
You need to verify it works.

Many dealership websites display a cookie consent banner — but the banner doesn't actually stop tracking tools from firing. Google Analytics, the Facebook pixel, and advertising platforms may be collecting visitor data the moment someone lands on your site, before they've agreed to anything.

This is the difference between a compliant website and a sham banner. Recent demand letters and lawsuits against dealerships have targeted exactly this scenario — tracking tools collecting visitor data without proper consent controls.

How to install

Up and running in 3 minutes

No technical knowledge required. For best evidence, run separate Fresh Visitor, Deny, and Accept tests in Chrome incognito.

1

Download the zip

Click the Download button above and save the .zip file to your computer.

2

Unzip the folder

Right-click the downloaded file → Extract All → choose your Desktop.

3

Open Chrome extensions

Type chrome://extensions in your address bar and press Enter.

4

Enable Developer Mode

Toggle Developer mode ON in the top-right corner of the extensions page.

5

Load the extension

Click Load unpacked → navigate inside the unzipped folder until you see manifest.json → click Select Folder.

6

Start auditing

The 🛡️ shield icon appears in your toolbar. Go to any dealership website and click it.
